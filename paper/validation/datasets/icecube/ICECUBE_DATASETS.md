# IceCube Neutrino Datasets

## HESE (High-Energy Starting Events) Data Releases

### HESE 7.5-Year Data Release (2021)
- **Events:** 102 HESE events
- **Time Range:** 2010-2017.5
- **Publication:** [PhysRevD.104.022002](https://doi.org/10.1103/PhysRevD.104.022002) / [arXiv:2011.03545](https://arxiv.org/abs/2011.03545)

**Download Links:**
- [GitHub Repository](https://github.com/icecube/HESE-7-year-data-release)
- [Official IceCube Page](https://icecube.wisc.edu/data-releases/2021/12/hese-7-5-year-data/)
- [DOI](https://doi.org/10.21234/4EQJ-BB17)

**Data Format:** JSON files with event data, Monte Carlo simulations, and Python analysis scripts

---

### HESE 12-Year Data Release (2023) - MOST RECENT
- **Time Range:** 2010-2022
- **Reconstruction:** DirectFit with updated glacial ice modeling
- **Presentation:** ICRC 2023

**Download Links:**
- [Harvard Dataverse](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/PZNO2T)
- [Official IceCube Page](https://icecube.wisc.edu/data-releases/2023/07/icecube-hese-12-year-data-release/)

### 10-Year Point-Source Data (2008-2018)
- **Events:** 1.13M neutrino candidates
- **Purpose:** All-sky point-source searches
- **Publication:** [arXiv:2101.09836](https://arxiv.org/abs/2101.09836)

**Download Links:**
- [NASA HEASARC Archive](https://heasarc.gsfc.nasa.gov/W3Browse/icecube/icecubepsc.html)
- [IceCube Announcement](https://icecube.wisc.edu/news/collaboration/2021/08/10-years-of-icecube-data-now-publicly-available-at-nasas-heasarc-archive/)

**Data Format:** FITS tables compatible with astronomy tools
